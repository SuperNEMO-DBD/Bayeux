Detailed Description

Needed for unit tests on portable archives.

Author
chris.nosp@m.tian.nosp@m..pfli.nosp@m.gers.nosp@m.dorff.nosp@m.er@g.nosp@m.mx.at

Header for testing portable archives with all of the serialization tests. Before use copy all hpp files from this directory to your boost folder boost_.../libs/serialization/test and run from there a visual studio prompt with b2 oder bjam -sBOOST_ARCHIVE_LIST=portable_archive.hpp

Note
Since portable archives version 5.0 we depend on program_options! Edit libs/serialization/test/Jamfile.v2 and change the requirements to : requirements {source}/boost/filesystem {source}/boost/program_options
